Fehler 503 - Die Seite ist momentan nicht verfügbar.
Bitte versuchen Sie es später nocheinmal.

Dies kann mehrere Ursachen haben:

  • Der Webauftritt der Charité wird überarbeitet.
  • Der Server der Charité ist überlastet.

Sie können folgendes tun, um die gewünschten Informationen dennoch zu erhalten:

  • Gehen Sie auf site:charite.de via www.google.de, um dort nach der gewünschten Seite zu suchen.
    Rufen Sie die dort zwischengespeicherte Version via 'Cache'-Link in den Suchergebnissen auf.

Unser Server hat diesen Fehler protokolliert und wir werden ihn gegebenenfalls reparieren.
Bei Fragen wenden Sie sich bitte an unseren Webmaster.



Error 503 - This site is currently not available.
Please try again later.

This error message could have the following reasons:

  • The website of the Charité is under construction.
  • The underlying webserver is too busy to fulfill your request.

You can do the following to get the information you are looking for:

  • Go to site:charite.de via www.google.de to search the page, you are looking for from there.
    Access the versions of our pages stored by them via the 'cache'-Link in the search results.

The error has been logged and we will fix it if neccessary.
Please contact our webmaster if you need further assistance.



Ошибка 503: запрашиваемая страница в данный момент недоступна.
Повторите попытку через некоторое время.

Это может быть вызвано следующим:

  • Идет обновление веб-ресурса Шарите.
  • Сервер Шарите перегружен.

    Найдите там временно сохраненную версию через 'Кэш'-ссылку в результатах поиска.
