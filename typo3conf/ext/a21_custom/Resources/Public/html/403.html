Fehler 403 – Sie sind nicht berechtigt, die angeforderte Seite zu sehen.

Dies kann mehrere Ursachen haben:

  • Die Seite ist nur charité-intern zugänglich, Sie besuchen uns aber aus dem Internet.
  • Die Seite benötigt eine Authentifizierung. Sie haben aber ein falsche Nutzername/Passwort-Kombination angegeben.
  • Sie haben versucht, ein Verzeichnis anzuzeigen.

Unser Server hat diesen Fehler protokolliert.
Sollte die Verweigerung des Zugriffs nicht richtig sein, wenden Sie sich bitte an unseren Webmaster.

 

 

Error 403 – You are not authorized to access this page.

This error message could have the following reasons:

  • This page is only accessible to Charité intranet users; you have tried to access this site via the Internet.
  • This page requires an authorization. You have used an incorrect user name/password combination.
  • You have attempted to access/open a directory.

The error has been logged and will be fixed if neccessary.
Should this access refusal be incorrect, please contact our webmaster  if you need further assistance.
